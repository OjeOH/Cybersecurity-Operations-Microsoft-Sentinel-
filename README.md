## 📖 About This Repository

This repository documents hands-on labs from the Cybersecurity Operations Management course, focused on cloud-native threat detection and incident response using **Microsoft Sentinel** (Azure SIEM/SOAR). Labs simulate real-world Security Operations Centre (SOC) workflows — ingesting data, writing analytics rules, triaging incidents, and hunting for threats.

## 🛠️ Skills Demonstrated

**Microsoft Sentinel (Azure SIEM/SOAR)**
- Provisioning a Sentinel workspace in Azure
- Deploying and configuring the Sentinel Training Lab solution
- Connecting playbooks via Logic Apps API

**Data Connectors & Ingestion**
- Azure Activity connector
- Microsoft Defender for Cloud connector
- Microsoft Defender Threat Intelligence connector

**Analytics & Detection Rules**
- Enabling and customizing rule templates
- Creating scheduled custom analytics rules with KQL queries
- Configuring Microsoft incident creation rules
- Reviewing Fusion rules (AI-based multi-stage attack detection)

**Incident Response**
- Assigning and managing incidents in Sentinel
- Using workbooks for investigation
- Identifying and documenting false positives
- Threat hunting with MITRE ATT&CK technique filters
- Adding IOCs to Threat Intelligence
- Creating bookmarks and promoting hunting results to incidents
